No More Running

A program designed to streamline communication between different workstations in a restaurant environment. Its high customizability enables adaptation to other industries, making it versatile and impactful.

About the Project

This project was designed and developed by Daniel Blake as a passion project. It leverages Python's capabilities alongside several powerful libraries to deliver a robust and efficient solution. Initially developed for restaurant workflows, its potential for other industries quickly became evident. With high customizability, it can cater to diverse use cases.

Key Features

  • Fast Communication: Lightweight communication over a brokered MQTT network.
  • Customizability: Easily adaptable to fit various industry needs.
  • Lightweight Security: Provides basic security to prevent misuse by laypeople (not designed for sensitive data).

Built With

This project utilizes the following libraries:

Standard Libraries

  • configparser: For reading and managing configuration files.
  • os: To handle file and directory operations.
  • time: For time-related functions.
  • threading: To enable concurrent execution of tasks.
  • hmac and hashlib: For cryptographic operations.
  • secrets: To generate secure tokens and passwords.
  • random: For random number generation.
  • tkinter: To create the graphical user interface (GUI).

Third-Party Libraries

  • paho.mqtt.client (v1.6.1): For MQTT messaging.
  • pygame (v2.6.1): For multimedia functionality, including audio playback.
  • Pillow (PIL, v11.0.0): To handle image processing and manipulation.
  • dateutil.relativedelta (v2.9.0.post0): For advanced date manipulations.
  • cryptography (v44.0.0): For secure encryption and decryption operations.
